# Introdução:
---

- O contexto é de umma atividade avaliativa em sala de aula. 
- Sendo seu objetivo exercitar a capacidade de utilzar algumas ferramentas de desenvollvimento com git e princcipalmente o Docker, gerando uam imagem é instanciando um container.
- O docker serve para criar e poder gerenciar container, que é basicmente "empacotar" aplicações com tudo que ela precisa rodar, sendo isso muito positivo para o uso do python visando que será a linguagem que usaremos em desenvolvimento Web.

## Relatar Atividades:
---

- Primeiramente criamso um fork do repositorio da atividade no GitHub
- Depois clonamos esse repositorio com Git
- Depois criamos os arquivos Python calculadora.py e alomundo.py
- Criamos o arquivo Dockerfile
- Criamos a imagem do fedora
- Damos o comando para o fedora instlar o python e outros comandos para execuçção
- Rodamos a imagem, assim, instanciando o container
- Fazemos alguns testes
- Upamos tudo no GitHub com o Git
